# Exercise Guide

Kuromaku Fit-Trak includes **30 exercises** across **3 categories** (10 per category). Each exercise has a stable internal ID, so renaming an exercise won't affect your historical data.

## Checkbox Behavior

- Checkboxes must be filled **left to right** in order
- Only the **next unchecked** box and the **last checked** box are tappable
- This prevents accidentally skipping sets

## Renaming Exercises

You can rename any exercise in **Settings > Manage Exercises**:
1. Tap the **pencil icon** next to any exercise
2. Enter your custom name
3. The exercise keeps its internal ID — historical data is preserved
4. Use **"Reset All to Defaults"** to restore original names

## Exercise Notes

While recording a workout, tap the **note icon** next to any exercise to add a short note (up to 100 characters). Notes are visible in history view and included in CSV exports.
